## Study Planner

Study Planner is a simple website that helps students organize their studies and keep track of upcoming exams.
Users can add subjects, set exam dates, create study tasks, track progress, and manage their study schedule in one place.

I made this so the students can focus on their studies

## Features

* Add new subjects
* Set exam dates for subjects
* Remove subjects
* Edit subject names
* Add study tasks for each subject ( very beneificial )
* Mark tasks as completed
* Progress tracking for each subject
* Exam countdown display
* Save subject data using Local Storage
* Clear all subjects at once


## Tech

* HTML
* CSS
* Javaaa


## How to Use

1. Open the website in a browser.
2. Enter a subject name.
3. Select an exam date.
4. Click the Add Subject....
5. Add study tasks to the subject.
6. Mark tasks as completed as you study.
7. View progress and exam countdown information.
8. Edit or remove subjects when needed.
